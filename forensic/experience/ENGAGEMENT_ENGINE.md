# ENGAGEMENT ENGINE — Spirit Revival Africa
## Systems That Bring Users Back Daily

---

## THE CORE TRUTH ABOUT DAILY ENGAGEMENT

People do not return to features. They return to:
1. **Obligations** (streaks, commitments, community expectations)
2. **Emotions** (feeling seen, belonging, spiritual growth)
3. **Habits** (the same time, same place, same routine)

Build all three.

## ENGINE 1: DAILY BREAD — MORNING ANCHOR

### Current State
Daily Bread exists in the backend and displays on the homepage. There is no:
- Notification
- Opt-in email delivery
- Sharing mechanism
- Community response

### Upgraded Daily Bread System

**The ritual:**
Every morning at **6:00 AM East Africa Time**, the Daily Bread triggers:

```
Channel 1: Email (opt-in)
  Subject: ✦ Today's Word — [Verse Reference]
  Body: Verse + 2-sentence reflection + Share button

Channel 2: WhatsApp (opt-in, priority for Africa)
  Format: [VERSE REFERENCE]
  "Verse text"
  — SRA Daily Bread

  (tap here to reflect with the community)
  https://spiritrevivalafrica.com/daily-bread

Channel 3: In-app notification (push, if PWA)
  "[Verse Reference] — Tap to read today's Word"
```

**Community response layer:**
Add a simple reaction strip below each Daily Bread:
```
[ 🙏 Praying This ]  [ 🔥 This Hit Different ]  [ 💬 Share a Reflection ]
  1,247                 834                          23 comments
```

The number of people "praying this" creates the daily engaged user count — visible, social, motivating.

**Weekly summary email (Sundays):**
```
This week, [username]:
→ You read the Word 4 days
→ 3 people prayed with you
→ Your most-impacted verse: Romans 8:1

[Read This Week's Final Reflection]
```

## ENGINE 2: PRAYER WALL — THE DAILY EMOTION LOOP

Prayer is the single most emotionally engaging action on the platform. Design a loop around it.

### Morning Prayer Assignment

Each morning, alongside Daily Bread, surface ONE prayer request to every logged-in user:

```
┌──────────────────────────────────────────┐
│  🙏  PRAY WITH THE MOVEMENT TODAY        │
│                                          │
│  "I'm praying for my father's salvation. │
│   He's 57. Please stand with me."        │
│                                          │
│  — James O., Nairobi                     │
│                                          │
│  [ I Prayed For This ] [ See All Prayers]│
└──────────────────────────────────────────┘
```

When a user clicks "I Prayed For This":
- The counter increases (+1 prayer partner)
- The person who submitted the request gets a notification: "12 people prayed for you today"
- The praying user gets a micro-affirmation: "Your prayer counts. 12 others prayed this this morning."

**This loop runs daily.** A new request each morning. 60 seconds. Deep emotional impact.

### Answer Reports — The Testimony Loop

When a prayer request gets answered:
- The requester can mark it "ANSWERED 🙌"
- A brief answer note is added
- It moves to the "Answered Prayers" archive (visible evidence God moves)
- An optional notification goes to everyone who prayed for it: "[Name]'s prayer has been answered. They said: [note]"

**This is the most powerful re-engagement trigger possible** — it closes a loop that users were emotionally invested in. They will return to see answered prayers.

## ENGINE 3: CONTENT DROPS — WEEKLY RHYTHM

### Publishing Schedule

Publish content on a **predictable cadence** so users know to return:

| Day | Content | Platform |
|-----|---------|---------|
| Monday | New article (teaching) | Website + Email + WhatsApp |
| Wednesday | Mid-week prayer focus (specific nation/topic) | Prayer wall + Email |
| Friday | Weekend word (shorter, devotional) | Website + WhatsApp |
| Sunday | Weekly reflection + answered prayers digest | Email |

**The key:** Users must be able to PREDICT when new content comes. Unpredictable publishing = users stop checking. Predictable cadence = habit formation.

### WhatsApp Content Delivery (Priority for Africa)

WhatsApp has 70–90% open rates in Africa vs 20–30% for email. This is the primary engagement channel.

**Implementation options:**

**Option A (Manual — start here):** Create a WhatsApp broadcast list. Add opt-in members. Send content manually 3x per week. Free. Immediate.

**Option B (Automated):** WhatsApp Business API. Can automate Daily Bread, prayer notifications, content drops. Costs ~$0.03–0.08 per message. At 1,000 users, ~$80/month.

**Option C (Chatbot):** WhatsApp bot that lets users type "PRAY" to get a prayer partner request, "VERSE" for today's scripture. No app needed.

**Start with Option A immediately.** Migrate to B when membership reaches 500+.

## ENGINE 4: GROUP ACTIVITY ALERTS

### Current State
Groups exist in the backend. There is no notification system for group activity.

### Upgraded Group Engagement

**Weekly Group Digest (every Sunday evening):**
```
Subject: 🔥 What's happening in Young Revivalists this week

3 new prayer requests
2 new members joined
1 member shared a testimony

[See What's Happening →]
```

**In-app indicators:**
- Show unread group activity count in nav badge
- Show "Last active: 2 hours ago" on group cards
- Show "3 new posts since your last visit" on group entry

**Group challenges (monthly):**
Each month, every group gets an optional "monthly challenge":
```
"This month's challenge: Pray for one unsaved friend every day.
 Share your updates in the group."
```

This gives groups a shared project, which is the single strongest driver of group retention in research on religious communities.

## ENGINE 5: STREAK SYSTEM (Light Touch)

### Philosophy
Heavy gamification is wrong for a spiritual platform. It makes prayer feel like a Duolingo lesson. But mild accountability — honoring consistency without punishing gaps — is deeply motivating.

### The "Revival Streak"

```
Your consistency this week:
M  T  W  T  F  S  S
🔥 🔥 🔥 ░  ░  ░  ░

3-day streak

"Keep showing up. Revival is built in small, faithful days."
```

**Rules:**
- Count is based on ANY engagement: reading, praying, or posting
- Shown on dashboard, not in notifications (non-intrusive)
- No punishment for breaking streak — just a reset ("New start!")
- At 7-day streak: small milestone badge ("Week of Fire")
- At 30-day streak: profile flair + community recognition

**Explicitly NOT included:**
- No emails saying "You broke your streak!"
- No guilt language
- No leaderboard by default (can be opt-in within groups)
